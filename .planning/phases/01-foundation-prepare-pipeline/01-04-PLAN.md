Clip discovery rules (PREP-03):

  - Match pattern: filename starts with `NN-` where N is a digit (one or more digits), the rest is any chars, extension `.mp4`. Regex: `/^(\d+)-.+\.mp4$/`. Files not matching are "skipped".
  - Sort matching files by the integer value of the prefix (NOT lexicographic — `10-clip.mp4` must come after `02-clip.mp4`).
  - If zero files match, throw `ClipDiscoveryError` whose message lists the contents of the folder under a heading `Skipped non-matching files:`.

  Signature:
    export function discoverClips(folderPath: string): { file: string; absPath: string }[];

ffprobe wrapper (PREP-04):

  - Spawn `ffprobe` (resolved from PATH) with args `['-v', 'error', '-show_entries', 'format=duration', '-of', 'default=noprint_wrappers=1:nokey=1', absPath]`
  - Parse stdout as float; round to 3 decimals (so floats are stable for hashing-free comparison)
  - On non-zero exit: throw `ProbeError` whose message contains the file name, the exit code, and the full stderr text
  - Implemented via `node:child_process` `execFile` wrapped in a Promise; do NOT use sync APIs

  Signature:
    export function probeDuration(absPath: string): Promise<number>;

Per-clip hash (PREP-07 input):

  Signature:
    export function computeClipSha256(absPath: string): Promise<string>;
  - Streams the file through `createHash('sha256')` and resolves with the 64-char lowercase hex digest (no prefix).

Manifest hash (PREP-07):

  Canonical input string (the contract Phase 2 MUST NOT break):

    canonical = folderName + '\n' +
                pairs.sort((a, b) => a.file.localeCompare(b.file))
                     .map(p => p.file + ':' + p.sha256)
                     .join('\n')

    manifestHash = 'sha256:' + sha256(canonical)

  - Music and render metadata are EXCLUDED from the hash.
  - Pairs are sorted by file name so iteration order does not flap the hash.
  - Newline separators are used so adjacent fields cannot collide via boundary ambiguity.
  - The function MUST be a pure deterministic function of (folderName, pairs).

  Signature:
    export function computeManifestHash(
      folderName: string,
      pairs: { file: string; sha256: string }[]
    ): string;
