Manifest schema (zod-validated; matches design spec, with one ADDITIVE deviation — per-clip `sha256` is exposed in the JSON so the hash is reproducible from the file itself):

  const clipEntrySchema = z.object({
    file: z.string().regex(/^\d{2,}-.+\.mp4$/),
    durationSec: z.number().positive(),
    sha256: z.string().regex(/^[0-9a-f]{64}$/),
  });

  export const manifestSchema = z.object({
    version: z.literal(1),
    kid: z.string().min(1),
    game: z.object({
      date: z.string().regex(/^\d{4}-\d{2}-\d{2}$/),
      opponent: z.string().min(1),
      scoreFor: z.number().int().min(0),
      scoreAgainst: z.number().int().min(0),
      result: z.enum(['W', 'L', 'D']),
    }),
    clips: z.array(clipEntrySchema).min(1),
    totalDurationSec: z.number().positive(),
    manifestHash: z.string().regex(/^sha256:[0-9a-f]{64}$/),
    // music + render blocks are NOT written by phase 1; phase 2/3 will extend the schema
  });

  export type Manifest = z.infer<typeof manifestSchema>;

      1. If `input.clips.length === 0`: throw `new ManifestError({ field: 'clips', reason: 'must contain at least one clip', remediation: 'add NN-*.mp4 files to the folder' })`
      2. Compute `manifestHash = computeManifestHash(input.folderName, input.clips.map(c => ({ file: c.file, sha256: c.sha256 })))`
      3. Compute `totalDurationSec = input.clips.reduce((s, c) => s + c.durationSec, 0)` then round to 3 decimals

      4. `writeFileSync(file, JSON.stringify(manifest, null, 2) + '\n', 'utf8')`

    Export type `PrepareResult = { skipped: boolean; reason: 'first-run' | 'hash-match' | 'hash-changed' | 'force'; manifest: Manifest; manifestPath: string }`.

    Export async `runPrepare(opts: { folderPath: string; channelsPath?: string; force?: boolean }): Promise<PrepareResult>`:
      1. `const absFolder = resolve(opts.folderPath);`
      2. `const folderName = basename(absFolder);`
      3. `const gameMeta = parseFilename(folderName);` // may throw FilenameError
      4. `const kid = resolveKidFromPath(absFolder, { channelsPath: opts.channelsPath });` // may throw KidPathError / UnknownKidError / ChannelsConfigError
      5. `const discovered = discoverClips(absFolder);` // may throw ClipDiscoveryError
      6. For each discovered clip in PARALLEL (Promise.all over the outer array): probe duration and compute sha256: `const [durationSec, sha256] = await Promise.all([probeDuration(c.absPath), computeClipSha256(c.absPath)]);` produce `{ file, durationSec, sha256 }`. (Probe and hash are independent — running them in parallel inside a single clip cuts wall-clock by ~half.)
      7. `const candidate = buildManifest({ folderName, kid, gameMeta, clips });`
      8. `const existing = readManifest(absFolder);` // may throw ManifestError on corrupted file
      9. Idempotency decision:
         - If `existing && existing.manifestHash === candidate.manifestHash && !opts.force`: return `{ skipped: true, reason: 'hash-match', manifest: existing, manifestPath: join(absFolder, MANIFEST_FILE_NAME) }` without writing
         - If `opts.force`: write, return reason `'force'`
         - If `existing && existing.manifestHash !== candidate.manifestHash`: write, return reason `'hash-changed'`
         - If `!existing`: write, return reason `'first-run'`
      10. `writeManifest(absFolder, candidate);`
      11. Return `{ skipped: false, reason, manifest: candidate, manifestPath }`
