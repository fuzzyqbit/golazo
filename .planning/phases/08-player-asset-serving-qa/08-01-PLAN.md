    Create web/src/lib/ui/rangeParser.ts as a PURE module — no node:*, no React, no next/* imports. Export:
    (1) Type `RangeRequest = { start: number; end: number }` — inclusive byte range
    (2) Function `parseRangeHeader(header: string | null | undefined, totalSize: number): RangeRequest | 'unsatisfiable' | null` where:
