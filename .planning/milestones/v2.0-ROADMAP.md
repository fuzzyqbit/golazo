## Overview

golazo v2.0 ships a localhost-only Next.js 16 web UI at `web/` for browsing and playing rendered episodes over the v1.0 CLI's filesystem-authoritative storage. Built as 4 horizontal phases: scaffold → discovery+cache → browse → player+QA. Read-only surface — CLI remains action authority for prepare/render/publish.

## Milestone Summary

**Decimal Phases:** none — Phases 5-8 shipped in order, no urgent insertions.

**Key Decisions:**
- D-01 workspace name `@golazo/cli` (not `@golazo/core`) — precise for v2.0 surface
- D-02 in-place layout (root stays CLI, `web/` sibling) — preserves all 387 v1.0 test fixture paths + `dist/cli/index.js` bin contract
- D-03 two-layer localhost defense — inline `HOST=127.0.0.1` env override + CLI `-H 127.0.0.1` flag + `instrumentation.ts` guard; the inline HOST override caught a Next.js subtlety (`-H` flag does NOT override `process.env.HOST`)
- D-04 fonts via cross-workspace `next/font/local` relative path — same TTF bytes as Remotion, no duplication, Turbopack accepted at execution time
- D-05 web tsconfig extends root + bundler resolution + `@/*` path alias
- D-16 sqlite driver `better-sqlite3 ^12.0` (sync API; production-grade native binding; macOS arm64 prebuilts)
- D-17 schema single denormalized `episodes` table, PK `manifest_hash`, covering index `(kid, date DESC, game_folder)`
- D-18 watcher 500ms per-folder debounce, canonical-absolute-path-keyed dedup
- D-19 invalidation via regex peek of `"manifestHash"` (no JSON.parse) + tracked-file mtime
- D-20 WarningBag in-memory only; startup-scan snapshot; never persisted
- URL contract `?sort=<key>.<dir>&kid=<filter>` with silent fallback + default-omission
- Server/client split — only `EpisodeList` is `'use client'`; pages are async Server Components calling `getDiscoveryRuntime()` server-side
- `notFound()` from `next/navigation` + custom `not-found.tsx` for PLAY-01 404 strategy
- Next.js 16 `searchParams` + `params` are `Promise<...>` — awaited server-side
- All cross-workspace imports via `@golazo/cli/dist/...` (never relative `../src/` traversal)
- Range parser inline (~107 LOC, single-range RFC 7233; multipart rejected)
- `VideoPlayer` `'use client'` island with NO node:* / `@golazo/cli` imports — hydration-safe
- Playwright Chromium-only — single-operator Mac; Firefox + Webkit deferred as `WEB-E2E-CROSS-BROWSER` v2.1 backlog
- Coverage gate enforced non-zero exit at <80%; exclusion cap ≤10 entries; included-surface floor ≥60%
